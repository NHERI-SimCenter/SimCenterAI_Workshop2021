TACC-Frontera¶

DesignSafe-ci is closely integrated within TACC. TACC designs and deploys the world’s most powerful advanced computing technologies and innovative software solutions to enable researchers to answer complex questions like those you are researching. For this workshop, and thanks to DesignSafe-ci, TACC is making available access to one of the fastest supercomputers in the world. To make use of these resources you need to be a good citizen while utilizing them.

If needed, you will be using the **Frontera** system and they provide a comprehensive set of usage notes.
